# Базовый пример (для платы BluePill-MIK32 ревизии №4)
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- В последовательный порт UART_1 отправляется строка со счётчиком.

***

## Сборка
```
make
```

## Заливка
!Внимание! В EEPROM контроллера должен быть залит загрузчик из ../ex_loader_3
```
make upload
```
