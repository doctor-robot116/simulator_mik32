# Пример работы с блоком GPIO_IRQ и поворотным энкодером.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Подключение поворотного энкодера к двум выводом одного порта (A -> PORT0.10, B -> PORT0.0).
- Проверено с энкодером PEC12R 24/24 (24PPR), режим FOUR3; проверено с энкодером EC11E 30/15 (30PPR), режим TWO03
- При изменении положения ручки энкодера в UART0 выводится текущая позиция и направление.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
