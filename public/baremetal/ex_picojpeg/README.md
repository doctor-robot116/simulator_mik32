# Пример работы с библиотекой picojpeg
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Выдача данных на экран через SPI_1 с использованием DMA.
- Замеряется скорость декодирования и вывода изображения 160x128 в формате JPEG (не поддерживается прогрессивное кодирование, и только целочисленное).
- Замеры для четырёх вариантов одной и той же картинки, с одинаковой "степенью сжатия", но с разным "цветовым разрешением". 4:4:4 - полное цветовое разрешение, 4:2:2h - половинное цветовое разрешение по горизонтали, 4:2:2v - половинное цветовое разрешение по вертикали, 4:2:0 - четверть цветового разрешения.
- Время декодирования и отображения каждого варианта картинки выводится в UART_0.

***

## Подключение экрана 160х128, контроллер ST7735:
- SCK   PORT1.2 (SCK1)
- LED   PORT1.0
- SDA   PORT1.1 (MOSI1)
- DC    PORT1.9
- RESET PORT1.8
- CS    PORT1.5

## Ещё заняты:
- NSS   PORT1.3 (NSS_IN) - к экрану не подключен.

***
