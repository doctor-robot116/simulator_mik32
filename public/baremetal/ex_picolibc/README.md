# Базовый пример для сборки с picolibc
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- В последовательный порт UART_0 отправляется строка со значением функции sin(X), где X меняется от 0 до 999 градусов.
- Почему в Makefile используется путь /usr/lib/gcc/riscv64-unknown-elf/12.2.0/rv32im/ilp32 для libgcc - потому, что в ядре MIK32 отсутствует набор "атомарных" команд, а libgcc с поддержкой компактной формы команд в моей сборке riscv64-unkonow-elf есть только в варианте rv32imac, так что на всякий случай использую rv32im.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
