# Пример работы с  I2C, получение данных от датчика BME280 (BMP280)
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- На экран (160х128, контроллер ST7735) выводятся данные о температуре, давлении (и относительной влажности, если датчик BME280).
- В последовательный порт (UART_0) раз в секунду выводятся данные от датчика.
- Этот пример только для сборки с размещением в SPIFI, так как в ОЗУ (и уж тем более в EEPROM) оно не влезет.

***

## Подключение экрана (SPI_1):
- LED   PORT1.0
- SCK   PORT1.2 (SCK1)
- SDA   PORT1.1 (MOSI1)
- DC    PORT1.9
- RESET PORT1.8
- CS    PORT0.2

## Ещё заняты:
- NSS   PORT1.3 (NSS_IN) - к экрану не подключен.

## Подключение датчика (I2C_1):
- SDA   PORT1.12
- SCL   PORT1.13

***

