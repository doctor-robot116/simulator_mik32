# Пример работы с кнопками
- По прерыванию от 32-битного (TIMER32_0) в регистр DAC0.VALUE загружается новое значение.
- Частоты формируются пятью DDS-генератороми с опорной частотой 50 кГц
- Ко входам GPIO_0: 10, 0, 8, 1, 2 (по порядку в разъёме на плате ELBEAR ACE-UNO) подключены кнопки, нормально разомкнутые, при нажатии замыкающие вход на "землю".
- Нажатие кнопки приводит к переключению (вкл/выкл) вывода сигнала соответствующего по номеру генератора. Сигналы нескольких включенных частот смешиваются.
- Длительное нажатие на кнопку запускает "автоповтор" нажатия (события нажатия с интервалом 64 мс).
- Сигнал выводится на выход PORT1.12 (DAC0).
- На выходе PORT1.13 (DAC1) формируется такой же сигнал, только "инвертированный", можно подключить высокоомный "наушник" непосредствено к выходам PORT1.12 и PORT1.13.

***

## Сборка
```
make
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```

## Заливка с загрузчиком ELBEAR
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```
