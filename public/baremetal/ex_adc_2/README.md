# Пример работы с АЦП
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Для АЦП настраиваются 4 канала (0, 1, 3, 4), значения из которых считываются циклически с частотой 50 кГц (т.е. "полный круг" по четырём канала идёт с частотой 12.5 кГц).
- В главном цикле выводятся значения (в порт UART_0) ото всех четырёх каналов сразу без какой-либо синхронизации.

***

## Подключение:
- Канал 0 АЦП - PORT1.5
- Канал 1 АЦП - PORT1.7
- Канал 3 АЦП - PORT0.4
- Канал 4 АЦП - PORT0.7

***
