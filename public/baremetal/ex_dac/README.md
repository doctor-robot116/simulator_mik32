# Пример работы с ЦАП
- По прерыванию от 32-битного (TIMER32_0) в регистр DAC0.VALUE загружается новое значение.
- Значения формируются DDS-генератором с опорной частотой 50 кГц
- Частота выходного "синусоидального" сигнала "качается" в диапазоне от 250 до 750 Гц, выход PORT1.12 (DAC0)
- На выходе PORT1.13 (DAC1) формируется такой же сигнал, на сдвинутый по фазе на +90 градусов

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
