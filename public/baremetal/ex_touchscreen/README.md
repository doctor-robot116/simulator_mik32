# Пример работы с тачскрином XPT2046, экран 320х240, контроллер ILI9341
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- С экраном общение через SPI_1 с использованием DMA.
- С тачскрином общение через SPI_0 без DMA.
- Пример выполняет калибровку экрана по четырём точкам, затем рисует жёлтый квадрат 5х5 в точке касания.
- Координаты касаний выводятся в UART_0.

***

## Подключение экрана:
- LED   PORT1.0
- SCK   PORT1.2 (SCK1)
- SDA   PORT1.1 (MOSI1)
- DC    PORT1.9
- RESET PORT1.8
- CS    PORT1.5

## Подключение тачскрина:
- SCK    PORT0.2
- MOSI   PORT0.1
- CS     PORT0.8
- MISO   PORT0.0
- PENIRQ PORT0.10

## Ещё заняты:
- NSS   PORT0.3 (NSS0_IN) - к тачскрину не подключен.
- NSS   PORT1.3 (NSS1_IN) - к экрану не подключен.

***

```

```
