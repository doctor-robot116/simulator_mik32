# Здесь лежат "драйверы" некоторых устройств
- Экран с контроллером ST7735 (SPI) - ../ex_spi_dma
- Экран с контроллером ILI9341 (SPI) - ../ex_spi_dma_2
- Экран с контроллером ST7789 (SPI) - использование то же, что и в ../ex_spi_dma* (с ST7789V3 работает, ST7789V2 - не работает)
- Датчик температуры.давления.влажности BME280/температуры.давления BMP280 (I2C) - ../ex_i2c
- NFC контроллер RC522 (SPI) - ../ex_mfrc522
- Трансивер Silabs SI4432/B1 (SPI) - ../ex_radio_tr, ../ex_radio_rc
- Контроллер резистивного 4-проводного тачскрина XPT2046, пример процедуры калибровки в ../ex_touchscreen
- Работа с устройствами по шине 1-wire с использованием UART - ../ex_onewire, ../ex_1wire_copy_key
- Поворотный энкодер - ../ex_encoder
