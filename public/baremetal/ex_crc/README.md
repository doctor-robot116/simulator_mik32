# Пример работы с блоком расчёта CRC.
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Считает CRC32 по разным "методам" на встроенном модуле CRC (https://ru.wikipedia.org/wiki/%D0%A6%D0%B8%D0%BA%D0%BB%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B8%D0%B9_%D0%B8%D0%B7%D0%B1%D1%8B%D1%82%D0%BE%D1%87%D0%BD%D1%8B%D0%B9_%D0%BA%D0%BE%D0%B4)
- Результат для тестовой строки выводится в UART0

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
