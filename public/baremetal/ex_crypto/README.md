# Пример работы с блоком CRYPTO и программным BLAKE2.
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Интерфейс для шифрования/расшифрования организован в виде трёх функций: do_crypto_init, do_crypto_update, do_crypto_final.
- Считает контрольные примеры AES-128/ECB/CBC/CTR с использованием блока CRYPTO.
- Считает контрольные примеры Кузнечик/ECB/CBC/CTR с использованием блока CRYPTO.
- Считает контрольные примеры Магма/ECB/CBC/CTR с использованием блока CRYPTO.
- Считает контрольные примеры для BLAKE2B/BLAKE2S.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
