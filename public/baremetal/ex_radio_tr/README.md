# Пример работы с трансивером SI4432 rev.B1, передача
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- "Общение" с SI4432 через SPI_1 
- Из последовательного порта UART_0 читается строка не длиннее 63 символов (завершение строки - символ LF/0x0A). Строка отправляется через радиоканал в виде одного пакета.
- Настройки трансивера: модуляция GFSK; Manchester OFF; центральная частота 434.1 МГц; девиация 6.25 кГц; скорость передачи 12.5 кБит/сек (индекс модуляции = 1.0); пакетная обработка с FIFO.
- "Калькулятор" настроек (с настройками для этого примера) лежит в файле ../docs/Si443x-Register-Settings_RevB1.ods
- Также "калькулятор" можно скачать с сайта SiLabs по ссылке https://www.silabs.com/wireless/proprietary/ezradiopro-sub-ghz-ics/device.si4432?tab=softwareandtools

***

## Подключение SI4432:
- SDO  PORT1.0
- SDI  PORT1.1
- SCLK PORT1.2
- SDN  PORT1.8
- nIRQ PORT1.9
- nSEL PORT0.2

## Ещё заняты:
- NSS   PORT1.3 (NSS_IN) - никуда не подключен

***
