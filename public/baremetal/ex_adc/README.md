# Пример работы с АЦП (и экраном 160х128, контроллер ST7735)
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- От АЦП набираются отсчёты в количестве, равным ширину экрана в пикселях, после чего выводится "гистограмма" по этим значениям.

***

## Подключение экрана:
- LED   PORT1.0
- SCK   PORT1.2 (SCK1)
- SDA   PORT1.1 (MOSI1)
- DC    PORT1.9
- RESET PORT1.8
- CS    PORT0.2

## Ещё заняты:
- NSS   PORT1.3 (NSS_IN) - к экрану не подключен.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
