# Пример работы с SPI/DMA: экран 320х240, контроллер ILI9341
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Выдача данных через SPI_1 с использованием DMA.
- На экран в цикле выводятся строки текста.

***

## Подключение экрана:
- LED   PORT1.0
- SCK   PORT1.2 (SCK1)
- SDA   PORT1.1 (MOSI1)
- DC    PORT1.9
- RESET PORT1.8
- CS    PORT1.5

## Ещё заняты:
- NSS   PORT1.3 (NSS_IN) - к экрану не подключен.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
