# Базовый пример для scmRTOS
- В качестве системного использован собственно системный таймер, прерывания от которого идут с частотой 1000 Гц.
- В обработчике таймера вызывается установка события (timerEvent).
- Один процесс (Proc0) ждёт установки события (timerEvent), на каждые 500 "сработок" события от таймера активируется событие (ledEvent) переключения светодиода.
- Второй процесс (Proc1) ожидает установки события (ledEvent) и переключает состояние светодиода.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
