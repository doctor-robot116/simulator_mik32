# Пример работы с NFC-чипом MFRC522 через SPI/DMA
- По прерыванию от 32-битного (TIMER32_0) таймера переключается светодиод.
- Прерывание от системного таймера увеличивает счётчик миллисекунд.
- Подключение к MFRC522 через SPI_1
- В порт UART_0 выводятся данные о поднесённой карте Mifare 1K/Ultralight

***

## Подключение MFRC522:
- MISO  PORT1.0 SPI MISO1
- MOSI  PORT1.1 SPI MOSI1
- SCK   PORT1.2 SPI SCK1
- CS    PORT0.2 GPIO выход
- IRQ   PORT1.9 GPIO вход
- RESET PORT1.8 GPIO выход

## Ещё заняты:
- NSS   PORT1.3 (NSS_IN) - никуда не подключен.

***

## Сборка
```
make
```

## Заливка
```
python3 ../tools/elbear_uploader.py ./out.hex --com /dev/ttyUSB0
```

## Заливка с загрузчиком ex_loader_2
```
make upload
```
