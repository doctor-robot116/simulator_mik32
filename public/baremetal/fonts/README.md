# Шрифты
Растровые шрифты со сглаживанием (8 цветов плавного перехода от цвета фона к цвету шрифта).  
Переменная ширина символов и фиксированная высота строки.  
Лежат в упакованном виде.  
Поддержка только 16-битного цвета.  
Используется кодировка utf-8.  

## Сборка утилиты для преобразования входных файлов в Сишные "исходники"
```
g++ font_2_src.cpp -Wall -Wextra -O2 -s -o font_2_src
```

## Файлы
1. font_2_src.cpp - исходники утилиты преобразования
2. font_bmp.h - заголовочный файл вспомогательных для отображения шрифтов
3. font_bmp.c - собственно исходники вспомогательных процедур
4. font_25_30.h и font_25_30.c - исходники для шрифта с высотой строки 30 точек и высотой символов 25 точек (чтобы нормально помещались всякие Й и у)
5. font_25_30_symbols.txt - содержит символы, доступные в шрифте font_25_30 (символ пробела есть всегда)
6. test32.h и test32.c - шрифт для примера

## Использование
1. Идём на https://snowb.org/ и настраваем нужный шрифт
2. Задаём белый цвет для символов и чёрный для фона
3. Обращаем внимание на размер шрифта и высоту строки - для нормального отображения символов с элементами выше (например Й) и ниже (например у) основного размера - высота строки должна быть больше размера шрифта
4. Экспортируем шрифт с параметрами "Font Name" = test_font, "File Name" = test_font и "Format" = "BMFont TEXT"
5. Распаковываем архив test_font.zip (внутри два файла: test_font.png с изображением символов и test_font.txt с описанием шрифта)
6. Открываем test_font.png в GIMP (ну или в Вашем любимом графическом редакторе), экспортируем как test_font.tga с отключенным "RLE compression" и "Origin" = Top left
7. В файле test_font.txt заменяем "Unnamed.png" на "test_font.tga"
8. Создаём исходники для шрифта командой ./font_2_src test_font.txt test_font.h test_font.c

## Дополнительно в директории utils/
1. test32.h и test32.c - шрифт для примера
2. test_font_bmp.cpp - тестовая утилита, выводит изображение шрифта в терминале в текстовом виде
3. Сборка: g++ test_font_bmp.cpp font_bmp.c test_font.c -I ../ -Wall -Wextra -O0 -g -o test_font_bmp
4. Использование: ./test_font_bmp "!"
